# Golden Mark Suite 🏆

منصة موحّدة لتطبيقات إدارة مبيعات Golden Mark

## هيكل المشروع

```
golden-mark-suite/
├── index.html              ← لوحة المدير الرئيسية (الـ Shell)
├── monthly-target/
│   └── index.html          ← تطبيق الهدف الشهري (بدون تعديل)
└── monitoring/
    └── index.html          ← نظام مراقبة المناديب (بدون تعديل)
```

## الميزات

- **لوحة مدير موحّدة** تجمع بيانات كلا التطبيقين
- **ملف شامل لكل مندوب** يعرض الأداء من التطبيقين
- **قائمة جانبية متجاوبة** — ثابتة على الكمبيوتر، drawer على الموبايل
- **بيانات حية من Supabase** — الهدف الشهري والشرائح
- **حالة ملفات المراقبة** من IndexedDB المحلي
